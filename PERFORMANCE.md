# 📊 Звіт продуктивності

## Тестове середовище

- **Машина:** MacBook Pro
- **OS:** macOS
- **Python:** 3.14
- **Ollama:** latest

## Результати

| Модель | Середній час (100 tokens) | RAM використання |
|--------|---------------------------|------------------|
| Llama2 7B | 2–5 сек | ~4.5 GB |
| Llama3 8B | 2–5 сек | ~5 GB |
| Mistral 7B | 2–4 сек | ~4.5 GB |

## Фактори впливу

- **CPU потужність** — більше ядер = швидша генерація
- **Кількість RAM** — моделі 7B потребують ~8 GB вільної пам'яті
- **Довжина промпту** — довший контекст уповільнює генерацію
- **Довжина відповіді** — час лінійно зростає з кількістю токенів

## Рекомендації

1. Для найкращої швидкості використовуйте Mistral 7B
2. Закрийте важкі програми під час роботи з LLM
3. Тримайте промпти короткими та конкретними
4. Використовуйте `stream: true` для кращого UX (відповідь з'являється поступово)
